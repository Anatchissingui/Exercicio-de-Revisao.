Exercício de Revisão: Conceitos Básicos de SQL, Java, C e C++

1. SQL (Structured Query Language):

  • O que é SQL e qual é o seu propósito no contexto de bancos de dados?

    • - SQL é uma linguagem de consulta estruturada usada para manipular e gerenciar dados em bancos de dados relacionais. Seu propósito é permitir a criação, consulta, atualização e exclusão de dados de forma eficiente.

  • Forneça um exemplo de uma consulta SQL simples:

    • sql
    -SELECT nome, idade FROM clientes WHERE cidade = 'Lisboa';

  • Explique o conceito de um banco de dados relacional?

    • Um banco de dados relacional armazena imformações em formato tabular,com linhas e colunas representando diferentes atributos de dados.
  • Cite pelo menos três tipos de dados que podem ser armazenados em um banco de dados relacional

    • 1. INT (inteiro)
    2. VARCHAR (cadeia de caracteres variável)
    3. DATE (data)

    3. Java:

  • Descreva brevemente a linguagem de programação Java:

    • Java é uma linguagem de programação orientada a objetos, independente de plataforma, projetada para ser simples, portátil e segura.
  • Mencione pelo menos três características-chave de Java:

    • 1. Portabilidade (Write Once, Run Anywhere - WORA)
    2. Orientação a Objetos
    3. Gerenciamento de Memória Automático (coleta de lixo)

    4. C e C++:

  • Quais são as principais diferenças entre as linguagens de programação C e C++?

    • - C++ é uma extensão de C que introduz a programação orientada a objetos, incluindo classes e herança.

  • Dê exemplos de situações em que cada uma dessas linguagens seria mais apropriada:

    • - C é frequentemente usado para programação de sistemas e desenvolvimento de hardware.
    - C++ é escolhido para desenvolvimento de software orientado a objetos, como jogos e aplicativos gráficos.

    7. Manipulação de Strings:


    Em Java, a classe String possui métodos para manipulação de strings, como concatenação, substring, etc.
    Em C, as strings são tratadas como arrays de caracteres. Em C++, a biblioteca string fornece métodos para manipulação.

    9. Alocamento de Memória:


  • Como a alocação de memória é tratada em C e C++?

    • Em C, a alocação dinâmica é feita com malloc() e free(). Em C++, utiliza-se new e delete.
  • Algumas diferenças significativas entre os dois.

    • C++ adiciona recursos de construtores/destrutores e elimina a necessidade de gerenciar explicitamente a memória (com exceções).

    10. Comandos SQL para CRUD:


  • Comandos SQL básicos para realizar operações CRUD (Create, Read, Update, Delete) em um banco de dados:

    • - Create: INSERT INTO tabela (coluna1, coluna2) VALUES (valor1, valor2);
    - Read: SELECT * FROM tabela WHERE condição;
    - Update: UPDATE tabela SET coluna1 = novo_valor WHERE condição;
    - Delete: DELETE FROM tabela WHERE condição;
